# Digital Humanities Summative Project: Do Female Funerary Stelae Demonstrate Gender Bias
Created for a Durham University module project. It aims to demonstrate data on female epigraphy and iconography from Greek funerary stelae to understand the extent of gender bias in the ancient world.
The Repositary Includes a Spyral notebook from voyant tools which contains two dataset analyses 1. Female Epigrapghy patterns from the AIO data base 2. Male Epighrapghy patterns from the AIO data base. Both are included within the notebook to allow viewers to compare and contrast the findings. Such findings are further developed within the required essay for the project. 
This repositrory also includes an attached recogito file with the recogito analysis findings in XML format. Users may repurpose this code to further define patterns from the selected AIO funerary iconography.
